Mongoid-rails

04 Aug 2015

This post is a guide in how to setup a rails project for mongoid instead of using the normal active records. I will start by saying if this is a new project save yourself the time of tracking down all the active record imports and create the project with the given command ‘rails new app –skip-active-record’

Gems Used:

  • rails 4.2.0
  • mongoid ~> 4.0.0
  • bson_ext
  • rspec-rails
  • mongoid-rspec ~> 2.1.0
  • rspec
